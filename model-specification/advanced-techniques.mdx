## Advanced Capabilities

### 1. Uncertainty Quantification

All models provide full posterior distributions:

**Python SDK:**
```python
from nimbus_pysdk import NimbusLDA, compute_entropy

clf = NimbusLDA()
clf.fit(X_train, y_train)

# Get probabilities and uncertainty
probs = clf.predict_proba(X_test)
entropy = compute_entropy(probs)

# Reject low-confidence predictions
high_conf_mask = probs.max(axis=1) > 0.8
reliable_predictions = clf.predict(X_test)[high_conf_mask]
```

**Julia SDK:**
```julia
using NimbusSDK

model = train_model(RxLDAModel, train_data)
results = predict_batch(model, test_data)

# Access uncertainty measures
confidences = results.confidences
entropy = results.entropy

# Quality-based filtering
high_quality = results.confidences .> 0.8
```

### 2. Online Learning & Adaptation

Update models incrementally without full retraining:

**Python SDK:**
```python
from nimbus_pysdk import NimbusLDA

# Initial training
clf = NimbusLDA()
clf.fit(X_initial, y_initial)

# Incremental updates
for X_batch, y_batch in new_data_stream:
    clf.partial_fit(X_batch, y_batch)
```

**Julia SDK:**
```julia
# Initial training
model = train_model(RxLDAModel, initial_data)

# Update with new data
updated_model = update_model(model, new_data)
```

### 3. Streaming Inference

Real-time chunk-by-chunk processing:

**Python SDK:**
```python
from nimbus_pysdk import StreamingSession
from nimbus_pysdk.data import BCIMetadata

metadata = BCIMetadata(
    sampling_rate=250.0,
    paradigm="motor_imagery",
    feature_type="csp",
    n_features=16,
    n_classes=4,
    chunk_size=125,
    temporal_aggregation="logvar"
)

session = StreamingSession(clf.model_, metadata)

for chunk in eeg_stream:
    result = session.process_chunk(chunk)
    # Use result.prediction, result.confidence

final = session.finalize_trial(method="weighted_vote")
```

### 4. Hyperparameter Tuning

Optimize model performance:

**Python SDK with GridSearchCV:**
```python
from sklearn.model_selection import GridSearchCV
from nimbus_pysdk import NimbusLDA

param_grid = {
    'mu_scale': [1.0, 3.0, 5.0, 7.0],
    'class_prior_alpha': [0.5, 1.0, 2.0]
}

grid = GridSearchCV(NimbusLDA(), param_grid, cv=5, n_jobs=-1)
grid.fit(X_train, y_train)

print(f"Best params: {grid.best_params_}")
best_model = grid.best_estimator_
```

### 5. Cross-Subject Transfer

Train on multiple subjects for better generalization:

**Python SDK:**
```python
from nimbus_pysdk import NimbusLDA, estimate_normalization_params, apply_normalization
import numpy as np

# Collect data from multiple subjects
all_X = []
all_y = []
for subject_id in subjects:
    X_subj, y_subj = load_subject_data(subject_id)
    all_X.append(X_subj)
    all_y.append(y_subj)

X_combined = np.vstack(all_X)
y_combined = np.concatenate(all_y)

# Normalize across subjects
norm_params = estimate_normalization_params(X_combined, method="zscore")
X_norm = apply_normalization(X_combined, norm_params)

# Train cross-subject model
clf = NimbusLDA(mu_scale=5.0)  # Higher regularization
clf.fit(X_norm, y_combined)
```
